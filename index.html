Our Air Quality Monitoring System (AQMS) is designed to track environmental conditions in real-time.

Equipped with the powerful MQ-135 gas sensor and the reliable DHT22 temperature and humidity sensor, the system collects vital air quality and weather data every few seconds.

Data is securely transmitted to the Firebase Realtime Database using an internet-connected ESP32 microcontroller. This dashboard provides a live, beautiful visualization of that data for easy monitoring.

The system aims to create safer, healthier environments by continuously detecting harmful gases, weather changes, and alerting users in real-time.
